# Hardware
- Arduino UNO (or Compatible)  
- PWM PC FAN  
- 12V Ac adaptor  

# Pin assignment  
|arduino|12V|FAN|
|---|---|---|
|GND|GND|GND|
|-|VCC|VCC|
|-|-|SENS|
|6|-|PWM|
