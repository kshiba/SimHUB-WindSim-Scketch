# SimHUB DisplayClientV2 for Easy Wind Sim
Sample VIDEO : https://twitter.com/k_shiba_rs/status/965105814934454273  

# Hardware
- Arduino UNO (or Compatible)  
- PWM PC FAN  
- 12V Ac adaptor  

# Pin assignment  
|arduino|12V|FAN|  
|GND|GND|GND|  
|-|VCC|VCC|  
|-|-|SENS|  
|6|-|PWM|
